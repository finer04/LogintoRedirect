# LoginToRedirect

某些开源平台没有认证功能或者无法满足等级保护要求，利用该程序可以给那些系统当作登录跳板平台。

目前还在开发中，还在慢慢填坑。前端只会 JQuery 可能代码质量非常一般。

## 初始化

访问 https://localhost:5000/setup-ui/ 即可完成部署
